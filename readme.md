# Proyecto con node.JS

## Primeros Pasos

Crear archivo:

```SH
index.js
```

inicializar package.json

```sh
npm init -y
```

instalar express

```sh
npm i express
```

Crear el gitignore

```sh
.gitignore
```
reiniciar server con observable

```sh
node --watch index.js
```
nodemos observa para reiniciar

```sh
npm i -D nodemon
```
